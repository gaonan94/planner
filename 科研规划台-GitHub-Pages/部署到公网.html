把规划台部署成
哪里都能打开的网站

推荐用 Netlify，最省事：把部署包拖进去，就会得到一个公网网址。

最快方式：Netlify 拖拽部署

  1. 打开 https://app.netlify.com/drop。
  2. 登录或注册 Netlify。
  3. 把 科研规划台-公网部署包.zip 拖进去。
  4. 等它上传完成，Netlify 会给你一个 https://... 网址。
  5. 手机、平板、学校电脑、家里电脑都打开这个网址即可。
如果你要保存跨设备数据，还要在应用“设置 → 云同步”里配置 Supabase。网站公网访问和数据同步是两件事。

也可以用 Vercel / Cloudflare Pages

  1. 新建静态网站项目。
  2. 上传这个文件夹或部署包。
  3. 构建命令留空。
  4. 输出目录填 . 或留默认静态目录。

上线后第一次要做什么

  1. 打开公网网址。
  2. 进入“设置 → 云同步”。
  3. 填 Supabase URL 和 anon key。
  4. 注册/登录邮箱账号。
  5. 点“同步”。

如果要用 OpenAI 一键生成建议

  1. 打开 https://platform.openai.com/api-keys。
  2. 登录 OpenAI 账号，创建一个 API Key。
  3. 回到规划台，进入 设置 → AI 配置。
  4. 接口地址填 https://api.openai.com/v1。
  5. 模型填 gpt-5.5。
  6. 粘贴 API Key，保存。
