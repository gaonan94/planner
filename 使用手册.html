科研健身英语规划台
简明使用手册

这个工具适合每天打开 3-5 分钟：先看今日概览，再快速记录，细节分别进入科研、健身、英语和科研 Idea 页面补全。

科研今日任务、短期项目、长期目标
健身训练内容、状态、体重
英语每周两次学习和 AI 建议
设置同步、标题、主题、备份

1. 每天怎么用

打开首页后，左侧只处理当天最常用的动作。

首页结构

  1. 先看左侧“今日概览”，确认今天有没有科研待办、健身记录、英语进度。
  2. 有临时想法时，用“快速记录”先记下来，不必马上整理。
  3. 需要补充细节时，进入顶部对应标签页。

2. 科研记录

科研页分成今日、短期项目、长期目标，适合把每天行动和长期方向接起来。

新增科研任务

  1. 任务写成可以行动的一句话。
  2. 层级选择“今日、短期项目、长期目标”。
  3. 备注里写进展、卡点和下一步。

科研 Idea

  1. 看到论文、做仿真、开会突然想到的点都放这里。
  2. 不用一开始判断好坏，先记录，再筛选。

3. 健身和英语

健身记录

  1. 训练日写具体内容，比如“力量 45 分钟 + 拉伸”。
  2. 休息日也可以记录成“休息”，方便看节奏。

英语学习和 AI 建议

  1. 先填写本次英语学习内容，尤其是话题、单词、句式。
  2. 在右侧 AI 区填 DeepSeek API Key，点“生成建议”。
  3. 如果请求失败，点“复制提示词”，粘贴到 DeepSeek 网页版。

4. 设置在哪里

主页看不到复杂设置。需要修改同步、标题、配色、备份时，进入顶部“设置”。

设置页结构

  1. 想跨设备使用：进入“设置 → 云同步”。
  2. 想改页面名称：进入“设置 → 标题设置”。
  3. 想换颜色：进入“设置 → 主题配色”。
  4. 想迁移数据：进入“设置 → 备份”。
建议：刚开始用时不必先配置云同步。先本地用几天，等记录方式顺手了，再去设置页打开同步。
